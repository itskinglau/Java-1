The purpose of project one is to help A small consulting firm assigns its employees to work on projects for its clients. Some of its 
employees work on some of the projects and some employees (such as HR staff) do not work on any project. An employee may work on multiple
projects. The company can hire new employees or start new projects. After an employee leaves the company or a project is completed, 
the corresponding records can be removed.
The Java requirements were:
1) develop a Java GUI application that reads employee and project information 
   from two text files and show their contents in two JList controls,
2) add a new employee or a new project,
3) delete unwanted employee(s) or project,
4) assign employee(s) to a project for a specified number of hours.
